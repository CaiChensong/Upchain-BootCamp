## 实践 SafeWallet 多签钱包

在 Safe Wallet 支持的测试网上创建一个 2/3 多签钱包。

- 往多签中存入自己创建的任意 ERC20 Token。
- 从多签中转出一定数量的 ERC20 Token。
- 把 Bank 合约的管理员设置为多签。
- 请贴 Safe 的钱包链接。
- 从多签中发起对 Bank 的 withdraw 的调用
